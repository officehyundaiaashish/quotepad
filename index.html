About This App

QuotePad is a smart, offline-first quotation maker — built for anyone who needs to create quick, professional estimates for anything.

Whether you're quoting for groceries, construction, events, or services — QuotePad lets you build itemized lists with rates, quantities, and auto-calculated totals — then export a clean PDF quotation in one tap.

Key Features

Smart List Maker
Add items from inventory with qty + rate
PDF Export
One-tap export to clean A4 portrait PDF
Master Inventory
Manage items, units & rates in one place
100% Offline
No internet required — data stays on device
Dark Mode
